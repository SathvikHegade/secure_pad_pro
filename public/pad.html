About SecureNote

Features

  • Password-protected notes
  • Auto-save every 2 seconds
  • File uploads (10MB max)
  • Preview PDFs and images
  • AI-powered summarization
  • Files expire in 24 hours

Privacy & Security

• Passwords hashed with bcrypt (10 rounds)
• Custom URLs for easy access
• Files validated for security
• No tracking or analytics
• AI powered by Google Gemini

How to Use

1. Create a note with a custom URL from the homepage
2. Set a secure password (bcrypt hashed)
3. Share the URL + password with collaborators
4. Type notes, upload files, get AI summaries
